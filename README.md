# NYC FDNY Emergency Response Time Analysis

Statistical analysis of emergency response times across New York City boroughs using public FDNY data.

## Overview
This project analyzes differences in FDNY emergency response times by borough to identify patterns and potential disparities. The analysis emphasizes statistical reasoning, visualization, and clear interpretation of results.

## Methods
- Data cleaning and preprocessing in R
- Exploratory data analysis and visualization
- Statistical comparisons across boroughs
- Interpretation of results and limitations

## Key Findings
- Emergency response times vary meaningfully across NYC boroughs
- Distributional differences suggest structural and geographic factors
- Results are observational and not causal

## Tools
- R
- RStudio
- ggplot2
- dplyr

The rendered HTML report (`fdny_analysis.html`) contains the full analysis and visualizations.
